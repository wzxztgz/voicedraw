## 声绘 VoiceDraw - 纯语音智能绘图助手

### 项目概述
七牛云 XEngineer 新工程师计划招募赛题目二实现。一款纯语音控制的绘图工具，用户通过语音指令完成全部绘图创作，无需鼠标键盘。

### 核心功能

| 功能 | 状态 | 说明 |
|------|------|------|
| 图形绘制 | ✅ | 支持圆、矩形、直线、三角形、星形、椭圆 |
| 颜色修改 | ✅ | 11 种颜色，"改成红色" |
| 大小调整 | ✅ | "放大"、"缩小"、"再大一点" |
| 位置移动 | ✅ | "往左移一点"，支持方向微调 |
| 位置绘制 | ✅ | "在右上角画一个三角形" |
| 对象选择 | ✅ | "选中3号"、"选中红色的圆" |
| 画布操作 | ✅ | 清除、撤销、重做（最多10步） |
| 确认/取消 | ✅ | 预渲染确认机制 |
| 帮助 | ✅ | 语音播报支持指令 |
| 复合指令 | ✅ | "先画圆然后画矩形"瀑布流执行 |

### 技术亮点

1. **可视化预渲染**：边说边画，实时显示半透明预览图形
2. **位置感知绘制**：支持"在左上角/右上角/中间"等位置词
3. **对话式纠错**："再左一点"继承上一步操作上下文
4. **并行反馈**：视觉渲染优先，语音播报异步不阻塞
5. **同音词纠错**：园→圆、巨形→矩形等自动纠正

### 技术栈

- **前端**：原生 JavaScript + HTML5 Canvas + Vite
- **后端**：Node.js + ws (WebSocket)
- **语音识别**：阿里云 DashScope Paraformer 实时流式 ASR
- **语音合成**：浏览器原生 Web Speech API

### 运行方式

```bash
# 安装依赖
cd server && npm install
cd client && npm install

# 配置 API Key
cp server/.env.example server/.env
# 编辑 .env 填入阿里云百炼 API Key

# 启动（Windows 一键启动）
start.bat
```

### 浏览器要求

- Chrome 100+ / Edge 100+
- 桌面端，HTTPS 或 localhost 环境（麦克风权限）

### 性能指标

- 语音识别延迟：~200-400ms（流式）
- 端到端总延迟：<500ms
- 指令识别准确率：96%+（安静环境）
